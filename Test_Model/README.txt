This directory provides an operation example of turbine disk, including finite element model (.cae) and result (.odb), assessment results (.xls and .png). 

'loadTemp.odb' and 'UnloadTemp.odb' is the temperature field applied in the 'Turbine_plane-2020.cae'. 

The assessment results are output from the plug-in. Job name of 'Test' considers material, model parameters, load, and geometry uncertainties. While job name of 'Testcase1' does not consider geometry  uncertainty.
